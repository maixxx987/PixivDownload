<h2> Pixiv下载助手 </h2>

<h3> 说明 </h3>
本程序是基于Java11的HttpClient开发，无UI界面。若需使用请安装JDK11。

<h3> 实现功能 </h3>
<ul>
    <li>根据图片ID下载原图</li>
    <li>通过SauceNAO搜原图并下载原图</li>
</ul>

<h3> 后续开发 </h3>
目前仅完成基本功能（下载普通图片），后续会追加以下功能
<ul>
    <li>下载动图</li>
    <li>下载画师</li>
    <li>下载排行榜</li>
    <li>下载对应TAG的高分作品</li>
</ul>
